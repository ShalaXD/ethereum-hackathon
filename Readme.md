# ETHWaterloo

This currently contains the starter code here: http://truffleframework.com/boxes/pet-shop

## To build it

1. Clone this repo
2. Start `testrpc` in a background tab
3. `truffle compile`
4. `truffle migrate`
5. `truffle test`
6. `npm install`
7. `npm run dev`
8. Go to localhost:3000. You should see dogs.


Happy Hacking!
